This is a brief explanation of all the steps that were taken during the Covid Data Exploration Project.

The main goal of this project is to exercise some abilities in Tableau as a method of data visualization.

COMPLETED: 12-05-2023

====================

Table of Contents (TOC):
1. Data Collection + Basic Manipulation
2. SQL Queries
3. Tableau Work

====================

1. Data Collection + Basic Manipulation

- Data collected from https://ourworldindata.org/covid-deaths
- Data opened in Microsoft Excel and the full dataset was divided into two portions, CovidDeaths and CovidVaccinations

====================

2. SQL Data Explorations

- Ran 4 different queries on the dataset to collect some information from the dataset to import into Tableau.
- All queries are available in CovidQuery2.sql
- All queries were ran then imported into Excel to save the result tables for us to use in Tableau.
- Any null values found in the tables were replaced with 0.
- Tables saves as Tableau Table 1-4 for each of the queries.

====================

3. Tableau Work

- Imported the 4 table into Tableau for dashboard creation.
- Created visual for Global Numbers.
- Created Bar Graph visual for Total Deaths Per Continent.
- Created Map Visual for Total Infection by Country.
- Created Line Graph + Forecast for Percent Population Infected for some major countries.
- Published Work
- Dashboard available at https://public.tableau.com/views/CovidDashboard_17019690386230/Dashboard1?:language=en-US&:display_count=n&:origin=viz_share_link
